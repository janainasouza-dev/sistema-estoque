

# Sistema de Controle de Estoque

## Descrição

Sistema web para gerenciamento de estoque, desenvolvido para permitir o controle de entrada, saída e cadastro de produtos. Ideal para pequenos negócios que necessitam organizar seu inventário de forma simples e eficiente.

## Funcionalidades

- Cadastrar novos produtos
- Editar informações de produtos existentes
- Excluir produtos do estoque
- Registrar entrada de itens
- Registrar saída de itens
- Consultar produtos cadastrados
- Visualizar status geral do estoque

## Tecnologias utilizadas

| Tecnologia | Versão |
|------------|--------|
| HTML | 5 |
| CSS | 3 |
| JavaScript | ES6+ |
| Node.js | 18.x ou superior |
| Banco de Dados | SQLite 3 |

## Como instalar

1. Clone o repositório:

   ```bash 
   cd pasta-do-projeto
   git clone https://github.com/seu-usuario/sistema-estoque.git

2. Acesse o diretório do projeto:
cd sistema-estoque

3. Instale as dependências:
npm install

4. Configure o banco de dados:
npm run setup-db

5. Inicie o servidor:
http://localhost:3000

Observações

Certifique-se de ter o Node.js instalado antes de prosseguir com a instalação.O banco de dados será criado automaticamente na primeira execução.
